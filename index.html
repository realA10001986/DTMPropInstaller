CircuitSetup/A10001986 DTM Prop Installer

This page is intended to ease initial firmware installation on fresh NodeMCU ESP32 boards for use in CircuitSetup/A10001986-designed DTM props such as Time Circuits Display, Flux Capacitor, SID, Dash Gauges or VSR. The installation works without installing any software apart from a USB driver. You don't have to worry about source codes, compilers, libraries and the likes.

Requirements:

  • NodeMCU ESP32 board
  • USB cable
  • Computer with Windows or MacOS
  • Google Chrome or Microsoft Edge browser
  • Driver for the USB-to-UART bridge on your NodeMCU ESP32 board, see below

Driver for USB connection

  • For ESP32 boards with a CP210x USB-to-UART chip, install this. On Windows, connect your ESP32, start the Device Manager, expand the "Ports (COM & LPT)" list and look for the port with the ESP32 name. On Mac, the port is named "cu.SLAB_USBtoUART".
  • For ESP32 boards with a CH340 USB-to-UART chip, Windows users may need to install a driver as well; see above on how to find out about the port. On Mac, the port is named cu.usbserial-XXXX (XXXX being some random number), no additional driver is required.

Let's get started:

  1. Please download the pre-compiled binary for the prop firmware you want to install; for all the DTM props, a pre-compiled binary is provided in the install folder of their respective repository. For direct download, click on one of the links below:
    • Time Circuits Display (A10001986 release); for CircuitSetup releases, see here.
    • Flux Capacitor
    • SID
    • Dash Gauges
    • VSR
  2. Now connect your NodeMCU ESP32 to your computer through USB and wait a few seconds. At this point, Windows users need to find out about the correct port, see above. Then click "Connect", select a port, and click Install DTMPropInstaller.
    Your browser is not supported. Please use Google Chrome or Microsoft Edge. HTTPS required, HTTP not supported
  3. After the upload has finished, disconnect the ESP32 from your computer, put it in/on your prop (if you haven't done that already), and power-up the prop. Next, connect your computer or hand-held to the WiFi network named "NP-AP". Navigate your browser to http://install.local or http://192.168.4.1 and click on "Update". On the page presented then, select the pre-compiled firmware binary to install, and click "UPDATE" to proceed. The firmware will be installed on your ESP32.

Future firmware uploads/updates can be done through the prop's WiFi-accessible Config Portal.
